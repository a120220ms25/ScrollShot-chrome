# ScrollShot 測試檢查清單

在發布或使用之前，請確認以下功能正常運作。

## 安裝測試

- [ ] 圖標已正確生成並放置在 icons/ 資料夾
- [ ] 擴充功能可以成功載入到 Chrome
- [ ] 沒有顯示任何錯誤訊息
- [ ] 擴充功能圖標出現在工具列

## 截圖功能測試

### C1: 可見區域截圖

- [ ] 點擊圖標 → 可見區域截圖可以運作
- [ ] 快捷鍵 Alt+Shift+1 可以運作
- [ ] 右鍵選單 → 可見區域截圖可以運作
- [ ] 截圖內容正確（視窗可見部分）
- [ ] 編輯器自動開啟

### C2: 完整網頁長截圖

- [ ] 點擊圖標 → 完整網頁截圖可以運作
- [ ] 快捷鍵 Alt+Shift+2 可以運作
- [ ] 顯示載入進度條
- [ ] 頁面自動捲動
- [ ] 截圖包含完整網頁內容（從頂到底）
- [ ] 捲動位置正確恢復
- [ ] 編輯器自動開啟

測試網站建議：
- 短頁面：https://example.com
- 中等頁面：新聞網站首頁
- 長頁面：維基百科文章
