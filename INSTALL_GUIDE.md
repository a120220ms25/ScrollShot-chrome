## 系統需求

### 最低需求
- ✅ Chrome 瀏覽器 88 版或更新版本
- ✅ 或其他 Chromium 核心瀏覽器（Edge、Brave、Opera）
- ✅ 任何作業系統（Windows、macOS、Linux）

### 檢查您的 Chrome 版本

1. 開啟 Chrome 瀏覽器
2. 在網址列輸入：`chrome://version/`
3. 按 Enter
4. 查看第一行的版本號碼

```
範例輸出：
Google Chrome: 120.0.6099.109 (正式版本) (64 位元)
                ^^^^^^^^^^^^ 這是您的版本號
```

如果版本號小於 88，請更新 Chrome：
- 前往 `chrome://settings/help`
- Chrome 會自動檢查並安裝更新

## 步驟一：準備圖標

ScrollShot 需要四個不同尺寸的圖標檔案。我們提供了簡單的生成工具。

### 選項 A：使用瀏覽器生成（推薦 ⭐）

**時間需求**: 1 分鐘

1. **找到圖標生成器檔案**
   ```
   ScrollShot/
   └── icons/
       └── generate-icons.html  ← 這個檔案
   ```

2. **開啟生成器**
   - 方法 1: 在檔案管理器中雙擊 `generate-icons.html`
   - 方法 2: 將檔案拖曳到 Chrome 視窗中
   - 方法 3: 在 Chrome 中按 `Ctrl+O`，選擇這個檔案

3. **生成圖標**
   ```
   您會看到：
   ┌─────────────────────────────┐
   │  ScrollShot 圖標生成器      │
   ├─────────────────────────────┤
   │  [16x16] [32x32] [48x48]    │
   │  [128x128]                  │
   │                             │
   │  [下載全部圖標]  [重新生成]  │
   └─────────────────────────────┘
   ```

4. **點擊「下載全部圖標」按鈕**

5. **移動檔案到 icons 資料夾**
   - 檢查您的「下載」資料夾
   - 您會找到 4 個檔案：
     - `icon16.png`
     - `icon32.png`
     - `icon48.png`
     - `icon128.png`
   - 將這 4 個檔案移動到 `ScrollShot/icons/` 資料夾中

6. **完成！** ✅

### 選項 B：使用 Node.js 生成

**適用對象**: 熟悉命令列的開發者

**時間需求**: 2-3 分鐘（含安裝依賴）

1. **確認已安裝 Node.js**
   ```bash
   node --version
   # 應該顯示 v14.0.0 或更新版本
   ```

   如果沒有安裝，從 [nodejs.org](https://nodejs.org/) 下載安裝。

2. **開啟終端機**
   - Windows: 按 `Win+R`，輸入 `cmd`，按 Enter
   - macOS: 按 `Cmd+Space`，輸入 `terminal`，按 Enter
   - Linux: 按 `Ctrl+Alt+T`

3. **進入專案目錄**
   ```bash
   cd /path/to/ScrollShot
   ```

4. **安裝依賴**
   ```bash
   npm install
   ```

   等待安裝完成（可能需要 1-2 分鐘）

5. **生成圖標**
   ```bash
   npm run create-icons
   ```

   您會看到：
   ```
   已建立 icons/icon16.png
   已建立 icons/icon32.png
   已建立 icons/icon48.png
   已建立 icons/icon128.png
   所有圖標建立完成！
   ```

6. **完成！** ✅

## 步驟二：安裝擴充功能

### Chrome / Brave 瀏覽器

**時間需求**: 2 分鐘

1. **開啟擴充功能管理頁面**

   三種方法任選一種：

   **方法 1**: 直接輸入網址
   ```
   在網址列輸入: chrome://extensions/
   按 Enter
   ```

   **方法 2**: 從選單進入
   ```
   點擊右上角三個點 (⋮)
   → 更多工具
   → 擴充功能
   ```

   **方法 3**: 使用快捷鍵
   ```
   Windows/Linux: Ctrl + Shift + Delete → 擴充功能標籤
   macOS: Cmd + Shift + Delete → 擴充功能標籤
   ```

2. **啟用開發人員模式**
   ```
   在頁面右上角找到「開發人員模式」開關
   ┌─────────────────────────┐
   │ 開發人員模式    [OFF] → │ 點擊切換到 ON
   └─────────────────────────┘
   ```

3. **載入擴充功能**
   ```
   啟用開發人員模式後，會出現三個新按鈕：

   ┌──────────────────┐
   │ 載入未封裝項目     │ ← 點擊這個
   ├──────────────────┤
   │ 封裝擴充功能      │
   ├──────────────────┤
   │ 更新             │
   └──────────────────┘
   ```

4. **選擇 ScrollShot 資料夾**
   - 在檔案選擇對話框中，找到並選擇 `ScrollShot` 資料夾
   - **重要**: 選擇整個資料夾，不是裡面的檔案！
   - 點擊「選擇資料夾」或「Select Folder」

5. **驗證安裝成功**
   ```
   您應該看到 ScrollShot 出現在擴充功能列表中：

   ┌─────────────────────────────────┐
   │ 📷 ScrollShot            v1.0.0 │
   │ 高效能截圖工具，支援長截圖...     │
   │ [詳細資料] [移除] [錯誤]         │
   │ 已啟用 ✓                        │
   └─────────────────────────────────┘
   ```

6. **固定到工具列（建議）**
   - 點擊 Chrome 工具列上的拼圖圖示（擴充功能）
   - 找到 ScrollShot
   - 點擊旁邊的圖釘圖示📌
   - ScrollShot 圖標會出現在工具列

7. **完成！** ✅

### Microsoft Edge 瀏覽器

步驟幾乎相同，只有一些細微差異：

1. 開啟 `edge://extensions/`
2. 開啟左下角的「開發人員模式」
3. 點擊「載入解壓縮的擴充功能」
4. 選擇 ScrollShot 資料夾
5. 完成！

## 步驟三：測試功能

安裝完成後，讓我們測試一下各項功能！

### 測試 1: 可見區域截圖（最簡單）

**時間**: 30 秒

1. 開啟任意網頁（例如 [google.com](https://google.com)）

2. 觸發截圖（三選一）：
   - **方法 A**: 按 `Alt + Shift + 1`
   - **方法 B**: 點擊工具列的 ScrollShot 圖標 → 選擇「可見區域截圖」
   - **方法 C**: 右鍵點擊網頁 → ScrollShot 截圖 → 可見區域截圖

3. 編輯器應該會自動開啟，顯示截圖

4. 測試編輯工具：
   - 點擊「箭頭」工具
   - 在截圖上拖曳畫一個箭頭
   - 點擊「文字」工具
   - 點擊截圖添加文字

5. 保存截圖：
   - 點擊「下載」按鈕
   - 或按 `Ctrl + S`

**✅ 如果能成功下載截圖，表示基本功能正常！**

### 測試 2: 完整網頁長截圖（核心功能）

**時間**: 1-2 分鐘

1. 開啟專案中的測試頁面：
   ```
   在 Chrome 中開啟: demo.html
   ```

   或者開啟任何長網頁（例如維基百科文章）

2. 按 `Alt + Shift + 2` 觸發完整網頁截圖

3. 您會看到：
   ```
   ┌─────────────────────────────┐
   │ 正在擷取完整網頁...          │
   │ ▓▓▓▓▓▓▓▓░░░░░░░░ 50%       │
   └─────────────────────────────┘
   ```

4. 頁面會自動捲動，請耐心等待

5. 完成後編輯器開啟，您會看到完整的長截圖

6. 測試縮放功能：
   - 點擊「放大」按鈕（+）
   - 點擊「縮小」按鈕（-）
   - 點擊「100%」重設

**✅ 如果能看到完整的長截圖，核心功能正常！**

### 測試 3: 馬賽克工具（隱私保護）

**時間**: 30 秒

1. 在編輯器中（從上一個測試繼續）

2. 點擊「馬賽克」工具（方格圖示）

3. 在截圖上拖曳選擇一個區域

4. 放開滑鼠，該區域應該變成馬賽克

5. 測試撤銷：
   - 按 `Ctrl + Z`
   - 馬賽克應該消失

**✅ 如果馬賽克效果正確，編輯功能正常！**

## 步驟四：自訂設定

### 修改快捷鍵

如果預設快捷鍵與其他程式衝突：

1. 前往 `chrome://extensions/shortcuts`

2. 找到 ScrollShot

3. 點擊鉛筆圖示編輯快捷鍵

4. 按下您想要的組合鍵

5. 點擊「確定」

**建議快捷鍵組合**:
```
可見區域: Ctrl + Shift + 1
完整網頁: Ctrl + Shift + 2
選定區域: Ctrl + Shift + 3
元素截圖: Ctrl + Shift + 4
```

### 調整權限

ScrollShot 需要以下權限才能正常運作：

- ✅ **讀取和變更您造訪的網站資料** - 用於截圖
- ✅ **儲存資料** - 用於暫存截圖
- ✅ **管理分頁** - 用於擷取畫面

這些都是正常的，不會收集您的個人資料。

## 常見問題

### ❓ 安裝後看不到 ScrollShot 圖標？

**解決方法**:
1. 點擊 Chrome 工具列的拼圖圖示
2. 找到 ScrollShot
3. 點擊圖釘📌固定到工具列

### ❓ 快捷鍵沒有反應？

**可能原因**:
1. 快捷鍵被其他擴充功能佔用
   - 前往 `chrome://extensions/shortcuts` 檢查
2. 在特殊頁面（如 chrome:// 頁面）無法使用
   - 這是瀏覽器限制，正常現象

### ❓ 某些網站無法截圖？

**正常現象**，以下網站受瀏覽器保護：
- Chrome 線上應用程式商店
- `chrome://` 開頭的頁面
- `about://` 開頭的頁面
- 銀行等高安全性網站

### ❓ 完整網頁截圖失敗或不完整？

**解決方法**:
1. 確保網頁完全載入
2. 關閉無限捲動功能
3. 手動捲動到底部後再截圖
4. 嘗試重新整理頁面

### ❓ 編輯器無法開啟？

**檢查項目**:
1. 彈出視窗是否被阻擋？
   - 檢查網址列的彈出視窗圖示
2. 按 F12 開啟 Console，檢查錯誤訊息
3. 重新載入擴充功能
4. 重新啟動瀏覽器

### ❓ 截圖品質不好？

ScrollShot 使用 PNG 格式，品質應該很好。如果品質不佳：
1. 檢查原始網頁品質
2. 確保縮放比例為 100%
3. 在高解析度螢幕上可能需要調整縮放

### ❓ 如何卸載？

1. 前往 `chrome://extensions/`
2. 找到 ScrollShot
3. 點擊「移除」
4. 確認移除

### ❓ 如何更新到新版本？

1. 下載新版本的檔案
2. 前往 `chrome://extensions/`
3. 找到 ScrollShot，點擊「重新載入」圖示
4. 或者移除舊版，重新安裝新版
